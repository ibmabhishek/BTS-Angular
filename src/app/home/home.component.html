Bug Tracking System

Using this software, a tester can create a request for a bug to be resolved by using the create bug option where the tester has to fill in the required fields like priority, status , type , project id and the date it was sumbitted on. This request is then to be reviwed by a devloper and once the developer acknowledges the request , can update about the bug by using the update option. Finally, we have an option of displaying the details of a specific bug or we can also generate a table to look into all the bugs.
